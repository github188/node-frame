1:安装 strongloop
  npm install -g strongloop 全局安装 strongloop类似于脚手架
2:初始化项目文件  命令: slc
3:创建模块 命令: slc loopback:model
4:data-source  :db(memory)
5:REST y
6:model's base class :PersistedModel  选择其他会出现401访问没权限
7:添加字段名字,数据类型
8:node . 启动 127.0.0.1:8080/explorer进入api页面,主要设定与数据库的接口
8:slc src 进入strongloop管理页面,可管理字段与数据库连接
9:安装对应的连接器 npm install loopback-connector-mongodb
